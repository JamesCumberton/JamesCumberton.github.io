Microstrategy Project
This project was based around the Business Intelligence tool Microstrategy. MicroStrategy is an enterprise business intelligence (BI) application software vendor. The MicroStrategy platform supports interactive dashboards, scorecards, highly formatted reports, ad hoc query, thresholds and alerts, and automated report distribution. Interfaces include web, desktop (for developers) and Microsoft Office integration. MicroStrategy Mobile also supports mobile BI. The reason behind choosing this project was I got to use this tool while I was on my 3rd year work placement and found it to be a very interesting and helpful tool. Eir were using it also as there new front end tool to their EDW(enterprise data warehouse) so like myself alot of the other employees were learning how to use it for the first time also. Within my project I done an evaluation on the tool and drew comparisons from various other tools sucha as Tableau etc. It was a challenging and intriguing project which I really enjoyed completing.
